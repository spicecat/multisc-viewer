## Description

Compare control, Alzheimer disease (AD), and Parkinson disease (PD) cell clustering and gene expression.

## Installation

```bash
$ git clone https://git.jasonxu.dev/JasonXu/plot-viewer.git
$ cd plot-viewer
$ yarn
```

## Running the app

```bash
# development
$ yarn run start
```

View app at [localhost:5000](http://localhost:5000)
